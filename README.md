# صحتنا (Sahatna) — منصة حجز المواعيد الطبية

> منصة رقمية تربط المرضى بالأطباء والعيادات في العراق، مشابهة لتطبيق Vezeeta ومكيّفة للسوق العراقي.

## 🚀 التشغيل السريع

نظراً لعدم توفر Node.js، التطبيق يعمل بدون أي خطوة بناء — يكفي تشغيل خادم محلي بسيط:

```bash
# باستخدام Python (متوفر على النظام)
python -m http.server 8000
```

ثم افتح المتصفح على: `http://localhost:8000`

أو ببساطة افتح ملف `index.html` مباشرة في المتصفح.

## 📁 هيكل المشروع

```
حجز/

│   ├── clinic.js       # منطق لوحة العيادة
│   └── admin.js        # منطق لوحة الإدارة
└── README.md
```

## 👥 أنواع المستخدمين

### 1. المريض (`index.html`)
- البحث عن طبيب حسب: التخصص، المدينة، السعر، التقييم، الجنس، نوع الخدمة
- عرض بروفايل كامل للطبيب (صورة، مؤهلات، خبرة، تقييمات موثّقة)
- حجز موعد فوري (تأكيد لحظي بدون انتظار موافقة)
- اختيار نوع الخدمة: زيارة عيادة / استشارة فيديو / زيارة منزلية
- الدفع عند الحضور للعيادة (Pay at clinic)
- تذكير تلقائي برسالة نصية قبل الموعد

### 2. العيادة (`clinic.html`)
**حسابات تجريبية:**
| العيادة | المستخدم | كلمة المرور |
|---|---|---|
| مركز الشفاء الطبي | `cl1` | `1234` |
| عيادة النور للتخصصات | `cl2` | `1234` |
| مستشفى الحياة الخاص | `cl3` | `1234` |

**الميزات:**
- عرض كل الحجوزات مع فلترة حسب الحالة والطبيب
- تقويم يومي/شهري للحجوزات
- إدارة أوقات دوام كل طبيب (يومياً، مدة الموعد)
- تأكيد/إلغاء/إكمال الحجوزات
- إرسال رسائل تذكير للمرضى
- إحصائيات (حجوزات اليوم، الإجمالي، الإيرادات)

### 3. الإدارة (`admin.html`)
**حساب تجريبي:** `admin` / `admin123`

**الميزات:**
- الموافقة/رفض العيادات الجديدة (خلال 48 ساعة)
- عرض كل الأطباء والحجوزات على المنصة
- إدارة التخصصات والمدن
- تحليلات شاملة (أكثر التخصصات طلباً، أكثر الأطباء حجوزاً، الإيرادات)

## 🇮🇶 تكييف السوق العراقي

- **المدن:** بغداد، البصرة، الموصل، أربيل، النجف، كربلاء، كركوك، السليمانية، وغيرها
- **الأسعار:** بالدينار العراقي (IQD)
- **الدفع:** "ادفع بالعيادة" كخيار أساسي (نظراً لمحدودية بوابات الدفع الإلكتروني)
- **اللغة:** عربي بالكامل مع واجهة RTL
- **التخصصات:** 12 تخصص طبي شائع في العراق

## 🛠️ التقنيات المستخدمة

| الطبقة | التقنية |
|---|---|
| Frontend | HTML5 + Tailwind CSS (CDN) + JavaScript (Vanilla) |
| الخطوط | Google Fonts (Cairo) |
| قاعدة البيانات | localStorage (Mock DB) |
| الصور | ui-avatars.com (صور رمزية للأطباء) |

> **ملاحظة:** هذا MVP يعمل بدون خادم خلفي (backend). البيانات تُحفظ في متصفح المستخدم. للإنتاج الفعلي، يُنصح بترحيل طبقة البيانات إلى PostgreSQL + Node.js/Django كما هو موضح في وثيقة المواصفات.

## 📊 البيانات الأولية (Seed Data)

- **12 تخصص** طبي
- **12 مدينة** عراقية
- **4 عيادات** (3 موافق عليها + 1 بانتظار الموافقة)
- **6 أطباء** ببيانات كاملة (صور، مؤهلات، أسعار، دوام)
- **5 تقييمات** موثّقة
- جداول دوام أسبوعية لكل طبيب

## 🔄 الترقية للإنتاج

لتحويل هذا MVP إلى منصة إنتاجية:

1. **Backend:** Node.js (NestJS) أو Django
2. **قاعدة البيانات:** PostgreSQL + Redis
3. **المصادقة:** JWT + OTP
4. **الإشعارات:** Firebase Cloud Messaging + بوابة SMS عراقية
5. **الدفع:** إضافة Zain Cash / FastPay / Qi Card
6. **الفيديو:** Agora.io أو Twilio Video
7. **الخرائط:** Google Maps API
8. **التطبيق:** Flutter أو React Native
